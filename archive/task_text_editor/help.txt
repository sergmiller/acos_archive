~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~Text Editor~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


Работа с текстовым редактором начинается либо с чтения текста из файла (имя файла указывается в аргументах функции main),
либо с создания нового текста, если файл оказался пустым, или не существовал, но его возможно создать по имени указанному в параметрах. 
Если имя файла не указано, то просто создаётся пустое множество строк.
После запуска доeлжно показываться приглашение ко вводу «editor:». В текстовом редакторе присутствуют группы команд.
Среди указываемому редактору команд могут встречаться комментарии. Комментарий начинается с символа '#' и следует до конца строки.
Если комментарий присутствует в добавляемой строке в текст, то такой комментарий просто является частью текста.
Комментарии, как и строчки состоящие только из пробельных символов должны  просто пропускаться редактором.


************************Группа команд просмотра текста****************************************

Команда print pages  поэкранного просмотра всего текста (сверху вниз и слева направо).
Просмотр следующего экрана по нажатию пробел. Выход из команды по нажатию 'q', либо исчерпании строк в тексте.
В случае запуска с перенаправленным выводом в файл, текст будет  выведен без приостановок на ожидание символа пробел.

Команда print range просмотра диапазона строк. После указания  самого имени команды следуют номера выводящихся строк. 
Номера разделены пробельными символами и отделены от имени команды пробельными символами, исключая перевод строки. 
Если второе число в диапазоне не указано, то считается что вывод до конца текста. Строки нумеруются начиная с единицы. 
Например: print range 10 20.


************************Группа команд вставки строк*******************************************

Команда insert after производит вставку строки или группы строк после N-ой. 
Сперва указывается номер строки (формат  по аналогии с предыдущей группой  команд), а затем в двойных кавычках сама вставляемая строка. 
Если номер пропущен вставка в конец текста. Если N равно 0 — в начало текста. 
Пример команды: insert after 3 “папа у Васи силён в \”математике\””. Кавычки могут быть экранированы символом '\'.
Для вставки группы строк используются три идущие подряд двойные кавычки, которые могут начинаться с новой строки, и заканчиваются так же 
тремя подряд двойными кавычками. В целом аналогично тому, как это устроено в языке программирования Python.


************************Группа команд редактирования строк************************************

Команда edit string Вставить/заменить символ в N-ой строке, начиная с M-ой позиции;
после указания позиций следует ключ, который может принимать  одно из 3 значений: -e (edit), -p (paste), -d(delete) 
и необходим соответствено для замены, вставки или удаления символа. 
Далее следуют номера строки и позиции символа, каждый из которых не может быть пропущен. 
После этого следует сам символ (если, конечно, не выбран режим удаления) в обычных кавычках, например:  edit string -e 3 7 's',
если его необходимо «защитить», используется '\' (например: ...'\n').

Команда replace ищет и заменяет подстроки в диапозоне строк. После имени команды идёт диапазон строк, так же
как и для команды print range (если диапазон не указан, то поиск и замена производятся во всём тексте), 
затем две строки в двойных кавычках, возможно каждая на новой строке. 
Первая из них строка образец, которая будет найдена в тексте, следующая строка, на которую будет производится замена. 
В заменяемой строке может встречаться экранированный перенос строки, тогда строку результат нужно будет разбить на несколько строк.
Вместо строки образца может присутствовать символ '^' - это означает, что строку нужно приписать в начало
каждой строки из диапазона, или символ '$' - это означает что приписать в конец.

Команда delete range - удаления строки. Указывается диапазон, аналогично команде print range. 
При этом первое значение не может быть пропущено и не может быть равным 0.

Команда delete braces позволяет удалить группы символов, расположенные между фигурными скобками { и }. 
Скобки тоже должны быть удалены. Предполагается, что скобки сбалансированы,  внутри каждой пары скобок могут быть другие скобки. 
В качестве аргумента указывается диапазон строк, если он не указан, то удаление производится во всём тексте.


************************Группа технических команд*********************************************

Команда exit осуществляет выход из текстового редактора.

Команда read имя_файла — открывает файл и в случае удачи замещает им все строчки. Имя файла не запоминается.

Команда open — аналогично read, за исключением того, что имя файла запоминается.
С помощью read и open могут быть считаны только существующие файлы.

Команда write — записывает строки в файл, если имя файла не указано, то запись происходит в «запомненное» имя файла, 
если его нет, то диагностируется ошибка. Если файл c указанным именем не существовал, то он создается.

Команда help  - печатает помощь для пользователя редактора. В режиме аналогичном команде print pages.
